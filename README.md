# SimuSched: CPU Scheduling Simulator

This is a Next.js application that simulates various CPU scheduling algorithms. It was built using Firebase Studio.

## Getting Started

To get started, take a look at `src/app/page.tsx`.

## Running Locally

To run the development server on your local machine:

1.  **Install dependencies** (only needed the first time):
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
